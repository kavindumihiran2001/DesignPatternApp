# Design Pattern Application

This is a Java Swing application developed using Apache NetBeans IDE 21.

## Features
- Payment processing system
- Strategy Pattern (Card and Cash payments)
- Singleton Pattern (Logger)

## Technologies Used
- Java
- Swing
- Apache NetBeans

## Design Patterns Used
- Strategy Pattern
- Singleton Pattern

## How to Run
1. Open project in NetBeans
2. Run the project
3. Enter name, amount, and payment method
4. Click "Process Payment"
